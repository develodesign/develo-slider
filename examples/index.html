Develo Slider

JQuery content slider

Simple, easy to use content slider plugin for jQuery that uses CSS3 animations.

Develo Slider is lightweight, only 4kb compressed and 2kb for the css.

Simple jQuery image slider

A simple jquery image slider of premier league football clubs* with controls that are only displayed when the slider is hovered.

* OK, maybe not all of them are premier league...

Show me 


<div id="slider-one">

	<img src="images/image1.png">

	<img src="images/image2.png">

	...

</div>

<script>

	$( '#slider-one' ).develoSlider({
		displayOnHover: true,
		items: $( '#slider-one img' )
	} );

</script>

Specify your own colour-schemes

You can uses the predefined colour classes to customise the colour of your content sliders or you can create your own colour classes in CSS and use them.

Show me 


<script>

var $sliderTwo = $( '#slider-two' ).develoSlider( {
	colourClass: 'blue',
	controls: {
		colourClass: 'black'
	},
	items: $( '#slider-two i' )
} );

var $sliderThree = $( '#slider-three' ).develoSlider( {
	colourClass: 'red',
	controls: {
		colourClass: 'white'
	},
	items: $( '#slider-three i' )
} );

</script>

jQuery Auto Slider

You can make your own html/image content slider that slides one slide at a time by ensuring the sliders container, and the move amount are the same as each slides width. By default the slider will not auto slide. However if you pass in a duration in as the autoSlide option, it'll cycle through all the slides.

Dynamically add items on the fly

You can access the slider object from anywhere and add or delete items on the fly. Nice feature if you are using ajax to populate the slider. (See the documentation for all the api options).

Show me 


<script>


var $slider = $( '#slider' ).develoSlider();

// Use the data jquery method to expose the slider API
var slider = $slider.data( 'develo-slider' );

// Populate the slider
$.ajax({
	success: function( data ){

		// OK so lets pretend data has some items that we want to spin over and add to the slider
		for( var i = 0; i < data.items.length; i++ ){

			var item = data.items[i]; // This could be an image? or any html content

			// Add it to the slider
			slider.addItem( item );
		}
	},
	url: 'your/url'
});

</script>
